# Task 2: Vulnerability Assessment with Nessus and OpenVAS

## Objective
Identify vulnerabilities on a target system using Nessus and OpenVAS vulnerability scanners.

## Tools Used
- Nessus
- OpenVAS
- Kali Linux

## Steps Performed
- Installed and configured Nessus/OpenVAS.
- Scanned the target machine for vulnerabilities.
- Exported and reviewed vulnerability reports.
- Captured screenshots of significant findings.

## Results / Findings
- Detected outdated software, misconfigurations, and critical vulnerabilities.
- Gained experience in vulnerability scanning workflows.

## Files Included
- Vulnerability scan reports (`.txt`/`.html`/`.csv`)
- Screenshots (`.png`)
